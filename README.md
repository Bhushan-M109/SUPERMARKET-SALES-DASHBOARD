# Super Market Sales Dashboard (Power BI)

## Overview
This project presents an interactive Power BI dashboard analyzing supermarket sales data from 2017–2020. The dashboard provides insights into sales performance, profitability, customer distribution, and product trends.

## Key Metrics
- Total Sales: 2.3M
- Total Profit: 283K
- Total Orders: 4.12K
- Total Quantity Sold: 30K

## Features
- Sales analysis by product sub-category
- Regional performance comparison
- Yearly and quarterly trend analysis
- Shipping mode distribution
- Customer geographic insights

## Tools Used
- Power BI Desktop
- Data Modeling
- DAX Measures
- Interactive Visualizations

## Files Included
- Dashboard.pdf → Static preview
- Dashboard.pbit → Reusable template

## Insights Derived
- Certain product categories generate significantly higher revenue.
- Regional performance varies widely.
- Seasonal trends impact sales and profit.

## Use Case
This dashboard can help business stakeholders:
- Identify top-performing products
- Monitor regional sales
- Optimize logistics strategy
- Improve profitability decisions
